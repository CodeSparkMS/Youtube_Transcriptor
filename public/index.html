YouTube Transcript API

This API provides YouTube video transcripts. Use the following endpoints:

GET /transcript/:videoId

Get transcript by video ID or URL

GET /transcript/1aA1WGON49E

POST /transcript

Get transcript by sending data in request body

POST /transcript
Body: {"videoId": "1aA1WGON49E"}

GET /health

Health check endpoint

GET /health

Note: This is a serverless API deployed on Netlify Functions.
